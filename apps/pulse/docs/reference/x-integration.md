# X Integration

Status: canonical

## Repo-Level Rule

- Product docs should describe the X workflow as the truth.
- Historical multi-platform ideas can be preserved as proposals or archived material.
- New work should not quietly reintroduce broader platform positioning without an explicit product decision.
