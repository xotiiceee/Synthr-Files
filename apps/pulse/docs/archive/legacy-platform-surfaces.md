# Legacy Platform Surfaces

Status: archived

Pulse currently presents itself as an X-only product.

The repo still contains some broader platform surfaces, such as:

- retained platform adapters in `src/platforms/`
- older multi-channel playbooks in `/playbooks`

These files are preserved because they may still be useful reference material or future expansion candidates, but they are not canonical proof of current product scope.

The canonical scope remains:

- X-specific product behavior
- hosted Pulse runtime
- Pulse's product-specific dependency on ClawNet
